My Favourite Projects:

BattleShip

My first real swing project, offering two-player gameplay.

Battleship is a game that dates back years, and here it has been beautifully recreated in Java.

MineSweeper

For the casual offline player looking to step up their game.

Created in Java using NetBeans, Minesweeper works in either 9x9 or 16x16 grids, with a flagging system that is a throwback to Windows XP.

Employmint

Created in Android Studio with the Indeed API.

A project meant to help youth find entry-level jobs. While Indeed and workopolis are intimidating and hard to navigate, Employmint is much easier on the eyes with similar features.
